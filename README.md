Here is the simple crawler app with the FE and BE implementation in TS/Node.js (require node v20.9.0)

To launch the app:
  - fork this repo to your local machine
  - thru the terminal install the npm packages in each folder 'crawler-fe' and 'crawler-server'
  - launch each instance with the command 'npm start' in each directory respectively
  - FE will be launched at localhost:3000, BE will be launched at localhost:8000
